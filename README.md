# Coherence Works Pathways


## Audio files

Playable SongBridge audio is wired through the `/audio` folder. Add MP3s using these exact filenames:

- `i-am-me.mp3`
- `joy-stays.mp3`
- `we-come-back.mp3`
- `many-ways-forward.mp3`
- `learnings-journey.mp3`
- `before-you-ask-them-to-learn.mp3`
- `the-space-to-learn.mp3`
- `learning-what-holds.mp3`
- `not-yet-still-here.mp3`
- `aun-no-sigo-aqui.mp3`

Once those files are uploaded, the embedded players will work automatically.
